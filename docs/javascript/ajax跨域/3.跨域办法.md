### **跨域解决方案**

1、 **JSONP** 通过jsonp跨域
2、 document.domain + iframe跨域
3、 **Hash** location.hash + iframe
4、 window.name + iframe跨域
5、 **postMessage** postMessage跨域
6、 **CORS** 跨域资源共享（CORS）
7、 nginx代理跨域 
8、 nodejs中间件代理跨域 (原理大致与nginx相同，都是通过启一个代理服务器，实现数据的转发)
9、 **WebSocket** WebSocket协议跨域
