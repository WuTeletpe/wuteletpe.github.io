### 索引
```python
out = torch.rand(2, 3)
print(out)
print(out[:, -1])

"""
第二个维度的最后一列

tensor([
        [0.3594, 0.3394, → 0.4230],
        [0.9116, 0.9131, → 0.0402]
])

tensor([0.4230, 0.0402])
"""


out = torch.rand(2, 3, 4)
print(out)
print(out[:, -1, :])
"""

第二个维度的最后一列

tensor([
        [
            [0.3510, 0.7428, 0.9492, 0.9024],

            [0.3938, 0.8071, 0.2525, 0.1012] #这一行
        ],

        [
            [0.8785, 0.7283, 0.3988, 0.1418],
            [0.3722, 0.5723, 0.2304, 0.3024],

        ]

tensor([
        [0.3938, 0.8071, 0.2525, 0.1012],
```

### 改变矩阵形状: view函数
```python
"""
view(*shape) → Tensor
- Parameters: shape (torch.Size or int...) – the desired size
- output: reshape the tensor to the shape  
- -1 代表该维度的大小还未确定,让 pytorch 自己计算决定
"""
a=torch.Tensor([[[1,2,3],[4,5,6]]])
print(a.view(3,2))

"""
tensor([[1., 2.],
        [3., 4.],
        [5., 6.]])
相当于就是从1，2，3，4，5，6顺序的拿数组来填充需要的形状.
"""

"""
如果您想得到如下的结果：
tensor([[1., 4.],
        [2., 5.],
        [3., 6.]])
就需要使用另一个函数了：permute()
"""
```
